# AI IDE：Trae

## 提示词：

<https://github.com/hassan-ak/wmd-ts-todo-list> 我是一个typescript入门学习，我现在要学习这个项目，帮我拆分一下这个项目，让后让我分模块进行学习。然后需要你帮助我从零开始复现这个项目，分阶段分模块实现该项目，且每个阶段的代码都可以正常运行，然后给出所有可能涉及到的指令和该指令的意义。将内容输出到markdown文件里

从第 1 阶段开始逐步讲解。然后再单独给我整理一份“逐步实现教程文档”，这个文档里要包括涉及到的代码的含义和所用到的指令和指令的含义。 <https://datawhalechina.github.io/easy-vibe/zh-cn/stage-2/backend/git-workflow/> 参考这个网址，每个阶段要用git和github进行版本管理，增加这个内容。

## 扩展：

0.可以增加一个标签功能，根据不同的标签来排序，或者时间来排序

1.可以在里面添加一个智能AI助手，使用工作流（workflow）进行分类，从而完成任务标签的添加。

2.使用AI进行模糊搜索，搜索所有的内容中符合的内容

3.用AI对单个任务进行拆分，做成plan的形式，可以进行一个自动的添加

4.将代办的内容的数据存放到本地，或者是supabase中

5.部署到网页上
